# st2195_assignment_3

This repository contains code for Practice Assignment 3 for ST2195 Programming for Data Science.

The task is to construct an SQLite database called `airline2.db` using Data Expo 2009 flight data and the supplementary files `airports.csv`, `carriers.csv`, and `plane-data.csv`. The repository includes R and Python code for building the database and reproducing the required SQL-based analyses.

The Data Expo 2009 dataset is available from the Harvard Dataverse
